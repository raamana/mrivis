mrivis¶

mrivis offers:

  • tools for comparison of spatial alignment of two 3D MRI scans (T1, T2, PET etc) using checkerboard plots and other comparison methods: Alignment check - usage
  • classes to develop customizable visualizations of 3D and 4D images - demo
